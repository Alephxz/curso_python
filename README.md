# Sistema de Gestión del Zoológico

## Descripción
Este programa permite gestionar y consultar información sobre los animales del zoológico.
Los datos se almacenan en archivos CSV y el sistema ofrece diferentes opciones de consulta
y administración.

## Instalación
1. Descomprimir el archivo en la ubicación deseada
2. Asegurarse de que los archivos `clases.csv` y `zoo.csv` estén en la misma carpeta que los scripts

## Ejecución
ejecutar el programa
